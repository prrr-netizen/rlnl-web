디스코드에서 끝내는 라이선스 관리, rlnl.

디스코드 전용 라이선스 관리 입니다. 실제 서비스에 적용 가능한 구조로 운영되고 있으며, 신규 기능은 점진적으로 추가되고 있습니다.

라이선스 상태 확인

결제 후 발급받은 라이선스 키를 입력하면, 유효 여부와 만료일을 확인할 수 있습니다.

Google 로그인 후 사용 가능합니다.

Features

자동 라이선스 발급

결제 완료와 동시에 라이선스를 자동 발급하는 구조로 설계되어 있습니다. 라니 MALL과 연동하여 별도 수동 작업 없이 키가 발급되도록 순차적으로 적용 중입니다.

디스코드에서 바로 인증

발급 받으신 라이선스를 알려 주세요. 인증 결과는 rlnl 웹 패널과 연동되어, 브라우저에서도 동일하게 확인할 수 있도록 합니다.

만료 & 역할 자동 관리

라이선스 만료 시 역할을 자동으로 제거하고 알림을 보내는 기능을 기준으로 설계되어 있습니다. 안정성을 충분히 검증한 뒤 실제 운영 서버에 단계적으로 적용할 계획입니다.

웹 대시보드

브라우저에서 라이선스를 한눈에 관리할 수 있는 대시보드를 제공 합니다. 키 상태, 사용자, 서버 정보를 직관적으로 확인할 수 있도록 함께 합니다.

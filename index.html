디스코드에서 끝내는 라이선스 관리, rlnl.

rlnl은 디스코드 전용 라이선스 관리 봇입니다. 지금은 테스트 단계로, 전용 테스트 서버에서 기능을 먼저 검증하고 있습니다.

Features

자동 라이선스 발급

결제 완료와 동시에 라이선스를 자동 발급하는 것을 목표로 개발 중입니다. 추후에는 상점과 연동하여 별도 작업 없이 키가 발급되도록 지원할 예정입니다.

디스코드에서 바로 인증

/license link 명령어 하나로 라이선스를 계정이나 서버에 연결하는 방식을 설계하고 있습니다. 인증 결과는 웹 패널과도 연동될 예정입니다.

만료 & 역할 자동 관리

라이선스 만료 시 역할을 자동으로 제거하고 알림을 보내는 기능을 개발 중입니다. 테스트 서버에서 먼저 안정성을 확인한 후 실제 서비스에 적용할 계획입니다.

웹 대시보드

브라우저에서 라이선스를 한눈에 관리할 수 있는 대시보드를 준비 중입니다. 키 상태, 사용자, 서버 정보를 직관적으로 확인하는 것이 목표입니다.
